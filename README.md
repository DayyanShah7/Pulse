\# Pulse



\*\*Pulse\*\* is a modern vape quitting and recovery companion app built with \*\*Kotlin\*\* and \*\*Jetpack Compose\*\*.  

The app is designed to help users reduce nicotine dependency through progress tracking, craving support, financial awareness, and motivational recovery tools.



\---



\## Overview



Pulse focuses on making the quitting journey feel less overwhelming by giving users a clean, supportive, and gamified experience.  

Instead of simply tracking days without vaping, the app helps users understand their progress, manage emergency cravings, and stay motivated through visual feedback and recovery milestones.



\---



\## Features



\- \*\*Personalized onboarding\*\*

&#x20; - Collects user spending and vape usage details

&#x20; - Builds a more meaningful recovery experience based on user habits



\- \*\*Progress tracking\*\*

&#x20; - Tracks vape-free progress

&#x20; - Displays estimated money saved

&#x20; - Helps users stay aware of their improvement over time



\- \*\*Emergency craving mode\*\*

&#x20; - Provides quick support during strong cravings

&#x20; - Includes calming options and defense strategies

&#x20; - Designed to help users pause before relapsing



\- \*\*Gamified recovery experience\*\*

&#x20; - Encourages consistency

&#x20; - Makes progress feel rewarding

&#x20; - Supports habit change through motivation and visual feedback



\- \*\*Modern Compose UI\*\*

&#x20; - Built using Jetpack Compose

&#x20; - Clean layout structure

&#x20; - Premium dark-themed interface
