Pulse

Break the habit. Regain control.

Pulse is a modern Android application designed to help users reduce nicotine dependency and stay consistent throughout their vape recovery journey.

The app focuses on creating a clean, motivating, and emotionally supportive experience through progress tracking, emergency craving support, financial awareness, and recovery-focused interactions.

✨ Features

• Emergency craving mode
• Recovery timer system
• Progress tracking
• Money saved tracking
• Personalized onboarding
• Modern dark-themed UI
• Smooth Jetpack Compose interface
• Motivational recovery experience

🛠 Built With

• Kotlin
• Jetpack Compose
• Material 3
• Android Studio
• Git & GitHub

📱 Screens Included

• Onboarding
• Home Dashboard
• Emergency Recovery Mode
• Progress Tracking
• Motivation & Recovery Flow

🚀 Future Improvements

• Room database integration
• Achievement system
• Recovery analytics
• Smart craving insights
• Cloud sync
• AI-powered recovery recommendations

Pulse was built with a strong focus on premium UI/UX, scalability, and creating a real-world wellness application experience rather than a basic tracker app.
